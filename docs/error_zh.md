## 错误设计全局概览
```
 __________________________________________________
| 解析     |  连接  |  编码  | sql语句  | 执行 |  解码 |
 --------------------------------------------------
   |          |        |       |      |      |--> 要求有数据，但是没有找到对应的行数据
   |          |        |       |      |      |--> 找不到字段
   |          |        |       |      |      |--> 字段类型不匹配
   |          |        |       |      |--> 违反了数据库约束，如果唯一键约束
   |          |        |       |      |--> 执行超时
   |          |        |       |--> 无法生成sql，例如mutation的所有字段都是none
   |          |        |       |--> 数据库解析sql后返回了语法错误
   |          |        |--> 类型不支持
   |          |        |--> sqlx在进行args add操作时触发了错误
   |          |--> 临时性连接失败, 例如无法从连接池获取连接
   |          |--> 永久性连接失败，例如配置错误，tls协议错误等
   |--> 比较操作符解析失败
   |--> 定位操作符解析失败
   |--> OrderBy结构体构建失败
```
